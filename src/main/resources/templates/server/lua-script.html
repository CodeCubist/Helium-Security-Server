Lua脚本API参考

服务器控制:

  • server:startServer() - 启动服务器
  • server:stopServer() - 停止服务器
  • server:reloadUsers() - 重新加载用户
  • server:reloadTokens() - 重新加载令牌
  • server:reloadActivationCodes() - 重新加载激活码
  • server:banIp(ip) - 封禁IP地址

状态检查:

  • server:isServerRunning() - 检查服务器是否运行
  • server:isWebManagerRunning() - 检查Web管理是否运行
  • server:getUserCount() - 获取用户数量
  • server:getTokenCount() - 获取令牌数量
  • server:getActivationCodeCount() - 获取激活码数量

工具函数:

  • util:sleep(ms) - 休眠指定毫秒
  • util:getTimestamp() - 获取当前时间戳
  • log(message) - 记录日志到服务器

事件函数:

  • onStart() - 脚本启动时调用
  • onStop() - 脚本停止时调用
  • onReload() - 脚本重载时调用
  • onSchedule() - 每30秒调用一次
  • onUserLogin(username, ip) - 用户登录时调用
  • onUserRegister(username, ip) - 用户注册时调用
